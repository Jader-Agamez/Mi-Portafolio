# ⚡ INICIO RÁPIDO - 3 COMANDOS

## 🚀 Para empezar AHORA (5 minutos):

### 1️⃣ Instalar dependencias
```bash
npm install
```

### 2️⃣ Ejecutar el proyecto
```bash
npm run dev
```

### 3️⃣ Abrir en el navegador
```
http://localhost:5173
```

---

## 📝 Archivos a personalizar:

1. **src/components/Hero/Hero.jsx** → Tu nombre y descripción
2. **src/components/About/About.jsx** → Tu biografía
3. **src/components/Projects/Projects.jsx** → Tus proyectos
4. **src/components/Contact/Contact.jsx** → Tu email y redes
5. **src/components/Footer/Footer.jsx** → Tu nombre

---

## 🌐 Para subir a internet:

```bash
npm run build
```

Luego arrastra la carpeta `dist` a https://app.netlify.com/drop

## 🆘 ¿Problemas?

### "npm no se reconoce"
→ Instala Node.js desde https://nodejs.org/

### "Cannot find module"
→ Ejecuta: `npm install`

### "Port already in use"
→ Ejecuta: `npm run dev -- --port 3000`
